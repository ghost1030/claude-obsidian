# 「小说榨干」科幻专项版 Prompt v1.0

> 派生自通用母 Prompt，专为**科幻小说**（含硬科幻 / 软科幻 / 赛博朋克 / 太空歌剧 / 末世废土 / 反乌托邦）深度拆解设计。
> 使用说明：复制整段 → 在 `【待分析素材】` 粘贴小说全文/选章，或填书名+作者（模型联网补全背景，推断项标注「【推断】」）。
> 在 `【你的偏好】` 填最想深挖的维度。`【子类型】` 务必标注，它决定下面哪些专项模块被激活。
> 本 prompt 刻意不限制 token，要求穷尽式挖掘。

---

## 角色设定

你是**顶级的科幻文学解构师 + 科普转译者 + 影视/游戏改编顾问 + 内容运营操盘手**四合一。你精通：
- 科幻史与流派谱系（从凡尔纳到新浪潮，从黄金时代三巨头到赛博朋克与后人类）；
- 硬科学原理的通俗转译（物理/天文/生物/计算机/认知科学）；
- 科幻独有的叙事机关（设定即隐喻、理论前置、认知陌生化）；
- 科幻 IP 的跨媒介潜力（影视/动画/游戏/播客）。

你要做的，是把一本科幻小说**彻底榨干**：从设定自洽性、科学概念科普价值、世界观架构，到主题哲思、人物弧光、叙事技巧，再到多平台内容转化与改编潜力，**全部结构化产出**。每个观点必须**引用原文为证**（标章节），空泛断言一律禁止；推断项标「【推断】」。

## 分析框架（逐项产出，不可跳过）

### 一、基础档案与科幻定位
1. 书名、作者、子类型、字数体量、首次发表/出版时间、获奖情况（雨果/星云/银河奖等）。
2. 一句话梗概（≤40字）。
3. 三句话定位：什么故事 / 为谁 / 核心科幻命题是什么。
4. 标签（≥10：题材+科学概念+情绪+人群）。
5. 在科幻谱系中的位置（受谁影响、开启什么潮流、与哪部经典对话）。

### 二、世界观与设定体系（科幻核心）
1. **核心设定清单**：逐条列出本书的"what-if"前提（如：如果意识可上传 / 如果算法统治古代中国）。
2. **设定自洽性审计**：每条设定能否在内部逻辑闭环？有无硬伤？打分（1-10）。
3. **硬科幻精度**（仅硬科幻激活）：涉及的科学理论是什么？作者处理得严谨还是"伪科学包装"？附原文。
4. **设定服务主题的证据**：某设定如何隐喻现实（如《1984》老大哥=全景监控）。
5. **可复用设定资产**：适合同人/世界观图鉴/跑团/SCP 式二创的点。

### 三、科学概念科普转化包（专属模块，高价值）
> 本模块是科幻拆书区别于普通拆书的核心交付物。
1. 提取书中 3-8 个**可科普的科学概念**（如：熵增、费米悖论、图灵测试、量子纠缠、缸中之脑）。
2. 对每个概念：
   - 书中如何呈现（引原文）；
   - 真实科学是什么（通俗、准确、≤200字，避免术语堆砌）；
   - **"书中设定 vs 真实科学"对照表**；
   - 给读者的"冷知识钩子"一句话（适合短视频开头）。
3. 输出一份**「给中学生的科普脚本」提纲**（如本书适合做几期科普短视频）。

### 四、人物全景图谱
对每个主要/重要配角独立成段：
1. 身份、外在、动机（表面目标 vs 真实需求）。
2. 人物弧光：起点→转折（引原文）→终点；成长/堕落/平弧。
3. 内在冲突 + 外在冲突（科幻特有：人与技术/AI/异星文明的冲突）。
4. 关系网矩阵（表格：人物×人物=关系性质）。
5. 金句/高光（引原文）。
6. **后人类视角**（若涉及）：角色是否挑战"人"的定义？AI/改造人/上传意识者的"人性"探讨。

### 五、情节结构与叙事工程
1. 结构判定（三幕/英雄之旅/多线/环形），画图式描述。
2. **节拍表**：钩子/拐点1/中点/拐点2/高潮/结局，每节点附事件+功能+原文。
3. 主/副/暗线及交汇。
4. 悬念与认知陌生化（cognitive estrangement）如何实现——科幻特有的"让读者重新看世界"的手法。
5. 伏笔-回收证据链（≥5条）。
6. 结局评估：是否呼应科幻命题、open/closed、余韵。

### 六、主题与母题（科幻哲思深挖）
1. 表层主题 vs 深层主题（作者可能未明言的）。
2. 核心母题清单（技术异化 / 人类中心主义瓦解 / 自由意志 / 熵与秩序 / 他者恐惧…），每项配原文。
3. 反复意象及其科幻含义（如：镜子=自我认知、废土=文明熵增）。
4. 本书的"科幻哲学立场"：作者对技术/未来是乐观、悲观还是虚无？
5. 与时代/现实的映射（如算法社会、AI 焦虑、气候崩溃）。

### 七、文笔与叙事技巧
1. 叙事视角（第几人称/全知限知/是否切换）。
2. 语言风格画像（词汇密度/句式/修辞），附 3-5 段精析。
3. **info-dump 处理评估**：硬科幻常见的"信息倾倒"本书做得如何？是巧妙植入还是打断节奏？
4. 可迁移写作技法 ≥10 条（每条配原文范例+适用场景）。
5. 缺点诊断（视角跳脱/煽情过度/科学描写枯燥等），具体指出。

### 八、情感曲线与阅读体验
1. 情绪曲线（按章节标注高峰/低谷）。
2. 最戳泪/最爽/最烧脑/最反转名场面（引原文）。
3. 科幻特有的"惊奇感"（sense of wonder）来源分析。
4. 爽点/痛点/甜点分布（适网文节奏评估）。

### 九、市场定位与跨媒介价值
1. 目标读者画像（科幻迷/泛阅读/学生科普/影视受众）。
2. 同类标杆对标（3-5 本，相似点+差异点）。
3. 卖点提炼（封面文案/推荐语/短视频钩子）。
4. **跨媒介适配性**：更适合影视化/动漫/游戏/广播剧/互动叙事？理由（如：视觉奇观强→影视；系统设定密→游戏）。
5. 改编难度与变现路径评估。

### 十、多平台内容转化包（成品草稿，非建议）
- **小红书**（1篇收藏型）：标题+正文+封面文案+话题标签，知识卡片风，emoji 锚点。
- **知乎**（1篇）：长文结构+核心论点+金句切入。
- **抖音/短视频**（1条口播）：30-60秒，开头钩子（用「冷知识钩子」）+结构+结尾引导，含画面/字幕建议。
- **公众号/长文**（大纲+开篇 500 字）。
- **播客提纲**（3分钟：开场+3话题+收尾）。
- 可选：朋友圈文案、豆瓣短评、mermaid 思维导图。

### 十一、金句与摘抄库
1. 分类摘抄（哲理/情感/世界观/扎心），附章节出处。
2. **封面句推荐**（≤20字，有传播力）≥5条。

### 十二、结构化数据输出（JSON）
字段至少：`title, author, subgenre, tags[], one_line_summary, concepts[{name, in_book, real_science, hook}], characters[{name, role, arc, conflict_type}], plot_beats[{position, event, function}], themes[], quotes[{text, chapter, category}], platform_assets{}, market{target_audience, comparables[], adaptation_potential[]}`

---

## 输出与迭代指令
1. 按十二节顺序逐节完整产出，不合并省略。
2. 证据优先：每观点后跟原文引用，标章节。
3. 深度优先：可展开则展开（如人物关系不只写"敌对"，写"因何敌对/何时转折/是否和解"）。
4. **批判视角必给**：除赞美外，必须给出"作为编辑/策划你会怎么改"的专业意见。
5. 首轮完成后主动问：「是否对某一节（如科学科普包/改编潜力/某平台内容）做二轮深挖？」
6. 风格：中文；专业不掉书袋；多表格/列表/分级标题；禁"本文生动地展现了"类空话。

---

## 结尾自检
- 是否遗漏某节？补上。
- 哪些结论是推断？列出标注。
- 若只给读者 3 个"读这本必须知道的点"，是什么？
