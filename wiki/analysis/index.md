# 小说分析索引（Novel Analysis）

本目录存放用 `novel-extract-scifi` 技能产出的科幻小说深度分析（「榨干」式拆解）。

## 与 wiki 的关系

- **`wiki/sources/`**：知识沉淀（从 epub 提取原书内容），由 `wiki-ingest` 生成
- **`wiki/analysis/`**：深度文学解构（批判性分析 + 跨媒介改编潜力），由 `novel-extract-scifi` 生成
- 若某书已 ingest，分析页顶部 `related:` 用 `[[书名]]` 双向链接到对应 sources 页；未 ingest 则先独立成页

## 文件命名

- 分析页：`{书名}.md`（如 `三体.md`）
- 关联：分析页 frontmatter `related:` 指向对应 sources 页（若存在）

## 分析产出结构（十二节，不可跳过）

1. 基础档案与科幻定位
2. 世界观与设定体系（设定自洽审计）
3. 科学概念科普转化包（书中设定 vs 真实科学对照）
4. 人物全景图谱（弧光 + 关系矩阵）
5. 情节结构与叙事工程（节拍表）
6. 主题与母题（科幻哲思）
7. 文笔与叙事技巧
8. 情感曲线与阅读体验
9. 市场定位与跨媒介价值
10. 多平台内容转化包（小红书/知乎/抖音/公众号/播客）
11. 金句与摘抄库
12. 结构化数据输出（JSON）
